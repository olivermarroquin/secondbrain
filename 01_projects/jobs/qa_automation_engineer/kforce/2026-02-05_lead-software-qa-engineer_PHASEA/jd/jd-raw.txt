Job Title: QA Automation Engineer

Must Have:
- Cypress
- TypeScript or JavaScript
- API testing (REST)
- CI/CD (GitHub Actions or Jenkins)
- SQL validation

Responsibilities:
- Build and maintain Cypress-based UI automation for web applications
- Create automated tests for REST APIs and validate responses
- Integrate automated tests into CI/CD pipelines
- Execute regression and smoke suites, report defects, and collaborate with engineers
